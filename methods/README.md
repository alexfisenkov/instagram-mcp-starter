# Три метода: обзор и как выбирать

Аналитику Instagram нельзя достать целиком из одного источника. Каждый метод закрывает свой слой данных. Ниже — что откуда берётся и в каком порядке подключать.

## Почему методов три

| Слой данных | Где лежит | Каким методом достаётся |
|-------------|-----------|-------------------------|
| Профиль, посты, охваты, показы, базовые инсайты, комментарии | Meta Graph API | **1 — Официальный API** |
| Весь контент профиля, медиафайлы (посты/Reels/карусели) для скачивания и базы, поля, которых нет в API | Веб-версия Instagram (ты залогинен) | **2 — Браузер** |
| Удержание по секундам, кривая просмотров, момент отвала аудитории, детальная аудитория Reels/Stories | **Только приложение на телефоне** | **3 — Телефон** |

Ключевая мысль: **самая ценная статистика для разбора сценариев — удержание и точки отвала — есть только в мобильном приложении** (метод 3). Официальный API её не отдаёт. Веб отдаёт больше, чем API, но тоже не всё. Поэтому для «полного ключа» нужны все три.

## Порядок подключения

```
Метод 1 (официальный API)   ← начни здесь: безопасно, стабильно, база для отчётов
      │
      ▼
Метод 2 (браузер)           ← добавь, когда нужен архив контента и скачивание
      │
      ▼
Метод 3 (телефон)           ← добавь, когда нужна глубина: удержание, отвалы, сценарии
```

Каждый следующий метод не заменяет предыдущий, а добавляет слой. Можно остановиться на любом.

## Сравнение

| | 1 · API | 2 · Браузер | 3 · Телефон |
|---|---------|-------------|-------------|
| Установка | одна команда | + браузерный тулинг | + Appium/adb (iOS нужен Mac) |
| Стабильность | высокая | средняя (Instagram меняет вёрстку) | средняя (зависит от телефона/ОС) |
| Риск для аккаунта | нет | низкий (только чтение своего) | средний (только чтение своего) |
| Уникальные данные | инсайты по API | архив + скачивание | **удержание, отвалы, аудитория** |
| Автоматизируемость | отличная | хорошая | ручная/полуручная |

## Общие правила для всех методов

1. **Только свой аккаунт.** Все три метода — про твой собственный Instagram и твои собственные данные.
2. **Только чтение.** Никаких лайков, комментариев, публикаций, подписок «руками агента». Read-only и по методам 2–3 в том числе — это защищает аккаунт от бана и тебя от случайных действий.
3. **Не сохраняй секреты как данные.** Куки, токены, session-строки браузера, UDID/серийники телефона — не выгружать в отчёты и не коммитить. В базу идут ссылки, метрики и медиа, а не «слепки» сессии.
4. **Уважай лимиты.** Быстрые массовые запросы (в любом методе) Instagram замечает. Читай спокойно, с паузами.
