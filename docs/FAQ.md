# FAQ

**Меня забанят в Instagram?**
Нет. Это официальный Meta Graph API: вы сами создаёте приложение в консоли Meta и сами выдаёте ему доступ. Никакого скрейпинга, эмуляции браузера или «серых» библиотек здесь нет. Лимиты запросов Meta контролирует сама (в худшем случае попросит подождать).

**Почему нельзя просто логин-пароль, зачем какое-то приложение Meta?**
Так устроен официальный доступ: Meta выдаёт данные только «приложениям» с ключами и явным согласием владельца аккаунта. Зато пароль никому не сообщается, а доступ отзывается одной кнопкой.

**Подойдёт личный (не бизнес) аккаунт?**
Нет — Meta API работает только с профессиональными (Бизнес/Автор). Переключение бесплатное, обратимое, на аккаунт не влияет: Instagram → Настройки → Тип аккаунта.

**Обязательно нужна страница Facebook?**
Для рекомендуемого пути — да (привязка занимает 2 минуты). Без страницы есть альтернатива — режим `instagram` («API setup with Instagram login»), агент знает про оба ([AGENT.md](../AGENT.md)).

**Что агент сможет делать с моим Instagram?**
Только читать: посты, статистику, комментарии. Публикация, ответы, сообщения — сознательно не реализованы.

**Мой пароль от Instagram кто-нибудь увидит?**
Нет. Пароль вы вводите только на настоящем сайте Meta при OAuth-входе. Серверу достаётся только токен с ограниченными правами.

**Что за App Review / проверка приложения? Мне это нужно?**
Нет. Проверка нужна, чтобы давать доступ ЧУЖИМ пользователям. Ваше приложение работает в режиме Development только для вас — этого достаточно.

**Почему токен «протухает»?**
Правило Meta: long-lived токен живёт ~60 дней. Продление — одна команда агенту («обнови токен Instagram»), делайте раз в месяц.

**Можно подключить несколько Instagram-аккаунтов?**
Да: каждому — свой профессиональный аккаунт в consent-экране и свой token-store (`META_TOKEN_STORE_PATH` в отдельном env + вторая запись MCP в клиенте). Продвинутый сценарий — попросите агента настроить по аналогии.

**Можно нескольким агентам сразу (Claude Code + Codex)?**
Да, без ограничений: сервер не держит постоянных соединений (в отличие от Telegram MCP), каждый агент просто запускает свой экземпляр. Токен у них общий из token-store.

**Данные уходят куда-то, кроме Meta?**
Нет. Сервер локальный, ходит только на graph.facebook.com / api.instagram.com. Прочитанное попадает в ваш диалог с Claude — как любой ваш чат с ним.

**Как полностью удалить?**
`uninstall.sh` / `uninstall.ps1` (см. README) + отзовите доступ приложения: Instagram → Настройки → Безопасность → Приложения и сайты.
