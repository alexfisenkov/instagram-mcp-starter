# Все инструменты (16)

Запоминать названия не нужно — говорите Claude обычным языком («покажи топ постов за месяц»), он сам выберет инструмент. Все инструменты **только читают** данные.

## Авторизация и сервис

| Инструмент | Что делает |
|-----------|------------|
| `meta_auth_status` | статус конфигурации и токена (токен показывается замаскированным) |
| `meta_scope_presets` | доступные наборы прав: readOnly / analytics / fullStandard |
| `meta_build_login_url` | строит официальную ссылку входа Meta OAuth |
| `meta_exchange_code` | меняет одноразовый код из redirect-адреса на long-lived токен (~60 дней) и сохраняет его локально |
| `meta_refresh_token` | продлевает токен ещё на ~60 дней |
| `meta_list_facebook_pages` | страницы Facebook, доступные токену, и связанные с ними Instagram-аккаунты |
| `meta_resolve_instagram_account` | находит и запоминает ваш Instagram user id (через страницу FB или напрямую) |

## Аккаунт и контент

| Инструмент | Что делает |
|-----------|------------|
| `meta_get_account_info` | профиль: имя, username, подписчики, число постов, аватар |
| `meta_list_media` | список постов/Reels с датами, подписями, ссылками, лайками и комментариями |
| `meta_get_media` | один пост подробно |
| `meta_get_top_media` | рейтинг последних постов по вовлечённости / лайкам / комментариям / дате |

## Аналитика (инсайты)

| Инструмент | Что делает |
|-----------|------------|
| `meta_get_user_insights` | метрики аккаунта: охват, показы, просмотры профиля и др. по дням/неделям |
| `meta_get_post_insights` | метрики конкретного поста/Reels: охват, показы, сохранения, взаимодействия |

## Комментарии

| Инструмент | Что делает |
|-----------|------------|
| `meta_list_comments` | комментарии к посту |
| `meta_get_comment_replies` | ответы на конкретный комментарий |

## Продвинутое

| Инструмент | Что делает |
|-----------|------------|
| `meta_raw_get` | произвольный read-only GET-запрос к разрешённым путям Graph API — для нестандартных задач |

> Чего здесь СОЗНАТЕЛЬНО нет: публикации постов, ответов на комментарии, сообщений в Direct — сервер read-only, чтобы его нельзя было использовать для спама и чтобы случайная команда агенту ничего не «наделала» в вашем аккаунте.
